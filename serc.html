The Robot Arm - hardware layer

// Hardware · In build

4-axis pick-and-place robotic system.

Our robot arm (called "MiniMEE") is designed for sub-millimeter placement accuracy. It uses NEMA 17, 14 and NEMA 8 stepper motors, TMC2209 silent drivers, ESP32 control and a 24V-powered vacuum end effector on a hollow-shaft wrist. To communicate with our software "PCB Workspace", commands are sent over from USB serial to a Flask backend that drives the web chat interface (named "Layla").

We have filed a provisional patent with USPTO on the AI-Assisted Robotic Arm System for Automated Component Assembly, Education and Remote Operations.
